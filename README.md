# Collect Bananas DeepQN Project

================================================================================

## Install


# Clone the repo

```
git clone https://github.com/idrisso4/Collect-Bananas-DeepQN
cd Collect-Bananas-DeepQN
```


## How to use the package

### To train the agent:

```
python main.py --train
```

### To Evaluate the agent:

```
python main.py --eval
```
